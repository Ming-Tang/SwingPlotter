# SwingPlotter

Plots the effects of popular vote swings for Canadian federal elections.

# Swing Model

The swing model used for seat projections is uniform swing based on
Canadian federal election 2015. See `data/` directory for dataset.

# Setup and Run

```
npm install
gulp
```
